### 训练要点

- **组间休息**：复合动作 90-120秒，孤立动作 60-90秒
- **核心法则**：渐进超负荷 — 3×15不费力→升一档回到3×10
- **每组次数**：8-15次（哑铃可稍高）
- **升级顺序**：加重量 → 加次数 → 加组数 → 慢速离心

## 🔧 遇到问题速查

### 体重3周不涨？
```
查薄荷App平均热量
  < 2300卡 → 每天加1杯牛奶+1个蛋（+200卡）
  ≥ 2300卡 → 训练有没有加重量？
    没加 → 下次加1-2.5kg
    加了 → 再等2周，外胚型增速慢
```

### 力量2周没进步？
1. 睡够7h了吗？ ← 最常见原因
2. 训练前2-3h吃了碳水吗？
3. 是不是太重动作变形了？降10%重量
4. 上次训练隔了48h+吗？

### 肌肉酸痛
- 训练后24-48h酸痛 → **正常**，该练照练
- 酸痛超4天 → 训练量过大，延后1-2天
- 关节尖锐刺痛 → **⚠️ 停练就医**

### 受伤了
- 局部受伤：停伤处，练其他部位
- 严重疼或肿：全停，就医
- 恢复时：先用50%重量起步

## ❌ 别犯这些错

| ❌ 错误 | ✅ 正确 |
|---|---|
| 先跑步减脂 | 你已经瘦了，直接增肌 |

| 只看体重 | 围度+力量+照片一起看 |
