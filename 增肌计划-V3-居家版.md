## 💪 第四部分：居家训练计划（V3 核心变更）

### 训练总原则（不变）

| 原则 | 说明 |
|---|---|
| **训练频率** | 每周 3-4 次（隔天练） |
| **每次时长** | 45-60 分钟（含热身和拉伸） |
| **训练方式** | 哑铃复合动作 + 自重训练 |
| **核心法则** | **渐进超负荷** — 每周比上周多一点 |
| **组间休息** | 复合动作 90-120 秒，孤立动作 60-90 秒 |
| **每组次数** | 8-15 次（哑铃/自重可稍高于器械） |

### 居家渐进超负荷策略（关键！）

> [!IMPORTANT]
> 你的哑铃有 4 个档位，渐进超负荷按以下优先级操作：

| 优先级 | 方式 | 你的实际操作 |
|---|---|---|
| 1️⃣ | **增加哑铃重量** | 🟢4kg → 🔵6.5kg → 🟡9kg（3 档递进） |
| 2️⃣ | **增加每组次数** | 3×10 → 3×12 → 3×15 |
| 3️⃣ | **增加总组数** | 3 组 → 4 组 → 5 组 |
| 4️⃣ | **升级动作变体** | 标准俯卧撑 → 窄距俯卧撑 → 背上放 2.5kg 配重片 |
| 5️⃣ | **慢速离心** | 3 秒下 + 1 秒停 + 1 秒上（增加肌肉紧张时间） |

**规则**：当你能用当前重量完成 3×15 且不觉得吃力时 → 升一档回到 3×10，重新爬。

## 🔰 第五部分：居家训练第 1 周指南（V3 新增）

> [!IMPORTANT]
> 第一周**不要急于上重量**，建立动作模式和训练习惯才是重点。

```
[第 1-2 天] 动作学习日
  → 看 B 站视频学习 A/B/C 三天的所有动作
  → 用 ⚪空杆（~1.5kg）模拟每个动作 10 次
  → 重点理解：哪个动作练哪个部位，应该感受到哪里发力

[第 3 天] 第一次正式训练（训练日 A）
  → 全部用 🟢轻档（~4kg），即使觉得很轻也不要加
  → 每个动作只做 2 组（不是 3-4 组）
  → 全程时间控制在 30 分钟以内
  → 训练后观察：48 小时内哪里酸痛（正常）

[第 5 天] 第二次正式训练（训练日 B）
  → 划船和硬拉可以试 🔵中档（~6.5kg），弯举保持 🟢轻档
  → 做满 3 组
  → 重点感受背部发力（背部是最难找到感觉的部位）

[第 7 天] 第三次正式训练（训练日 C）
  → 高脚杯深蹲试 🟡重档单只（~9kg），箭步蹲用 🟢轻档×2
  → 腿部训练后 48-72 小时酸痛是正常的
  → 深蹲和箭步蹲的膝盖方向要注意（与脚尖同向）

[第 2 周起] 正式按计划执行
  → 按训练表的「起始重量」执行，能做 15 次就升一档
  → 开始记录每次训练的重量和次数（tracker-app 或备忘录）
```

### ⚡ 居家 → 健身房过渡信号（什么时候该去健身房）

> [!IMPORTANT]
> 当出现以下**任意 2 个**信号时，说明居家训练效果开始递减，应考虑过渡到健身房：

| 信号 | 你的具体判断标准 |
|---|---|
| 🔴 **哑铃重量到顶** | 划船、卧推用 🟡9kg×2 能做 15+ 次；深蹲用 🔴16.5kg 能做 15+ 次 |
| 🔴 **力量 2 周没进步** | 已尝试增加次数/组数/慢速离心，仍无进展 |
| 🟡 **体重增长停滞** | 排除饮食问题后，连续 3 周体重不变 |
| 🟡 **训练感觉太轻松** | 训练结束后不觉得累，没有肌肉泵感 |

## ⚠️ 第十一部分：居家训练的诚实分析

### 居家训练能增肌吗？

**答案：能，但有上限。**

| 维度 | 居家（哑铃+自重） | 健身房 |
|---|---|---|
| 前 1-2 个月效果 | ⭐⭐⭐⭐ 几乎一样好 | ⭐⭐⭐⭐⭐ |
| 3-6 个月效果 | ⭐⭐⭐ 开始受限 | ⭐⭐⭐⭐⭐ |
| 渐进超负荷上限 | ~9kg/只（你的配置），约 4-8 周到顶 | 几乎无限 |
| 动作多样性 | 有限但足够起步 | 非常丰富 |
| 安全性 | ⭐⭐⭐⭐⭐ 更安全 | ⭐⭐⭐⭐ |
| 便利性 | ⭐⭐⭐⭐⭐ 省时省钱 | ⭐⭐⭐ |

> [!IMPORTANT]
> **结论**：对于零基础的你，前 1-2 个月居家哑铃训练**完全足够**。
> 
> 新手福利期（Newbie Gains）的本质是肌肉从未受过刺激导致的超敏反应，**与器械高级程度无关**。
> 哑铃深蹲照样能激活你的腿部肌肉，哑铃划船照样能练厚背部。
> 
> 但要有清醒的认知：**当哑铃重量不够用时，必须过渡到健身房**，否则增肌将陷入停滞。
