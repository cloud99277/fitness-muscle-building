## 哑铃重量速查

| 档位 | 单只重量 | 适合动作 |
|------|----------|---------|
| 🟢 轻档 | ~4kg | 侧平举、弯举起步 |
| 🔵 中档 | ~6.5kg | 肩推、划船起步 |
| 🟡 重档 | ~9kg | 卧推、划船、硬拉 |
| 🔴 满载 | ~16.5kg | 高脚杯深蹲（双手） |

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 降级训练速查

```
正常     → 完整 45-50 分钟
轻度累   → 少做 1 个辅助动作
很累/<6h → 2-3 个主动作，每组 2 组，20-30 分钟
关节痛   → 停练
```
