## 💪 第三部分：训练计划

> [!WARNING]
> **⚠️ 零基础者请先阅读第四部分 B（私教建议）再继续**（v1→v2 新增，来源：🔴2）  
> 没有私教指导直接做杠铃深蹲/卧推等动作，姿势错误极易受伤。建议至少预约 1 节私教体验课后，再按本训练表执行。

### 🔰 零基础第 1 周操作指南（v1→v2 新增，来源：🔴2）

**第一次进健身房不要直接开始训练计划**，按以下流程来：

```
[第 1 天] 熟悉环境（20 分钟）
  → 找到每个训练表里的器械在哪里
  → 观察其他人如何调节器械
  → 不做正式组，只做动作练习（空杆/空手）

[第 2-3 天] 轻重量试水
  → 选比你想象中还要轻 50% 的重量
  → 每个动作只做 2 组（不是 3 组）
  → 做完后观察：肌肉酸痛是否在正常范围（次日酸是正常的）

[第 4-7 天] 正式开始 A/B/C 计划
  → 确认重量选择合适后，按训练表执行
  → 第 1 周目标：把动作做标准，不追求重量
```

> [!TIP]
> **起始重量如何选**：能做 15 次但第 12-15 次开始费力的重量，就是合适的起始重量。完全不费力 = 太轻，做不到 8 次 = 太重。

### 训练总原则

| 原则 | 说明 |
|---|---|
| **训练频率** | 每周 3-4 次（如一三五或一二四五） |
| **每次时长** | 60-75 分钟（含热身和拉伸） |
| **训练方式** | 以复合动作为主（同时练到多个肌群） |
| **核心法则** | **渐进超负荷** — 每周比上周多一点（重量 / 次数 / 组数） |
| **组间休息** | 复合动作 90-120 秒，孤立动作 60-90 秒 |
| **每组次数** | 8-12 次（增肌最佳区间） |
| **有氧训练** | 每周最多 1-2 次，每次 20 分钟（瘦人增肌少做有氧！）|

### 推荐方案：三天全身训练（初学者最优解）

训练日安排：**周一 / 周三 / 周五**（或任意隔一天练一天的安排）

## 🏋️ 第四部分（B）：关于健身房和私教的建议

### 是否需要去健身房？

**结论：强烈建议去健身房，至少在前 3-6 个月。**

| 维度 | 健身房 | 家里 |
|---|---|---|
| **器材** | 杠铃、哑铃、器械一应俱全 | 只有自重或少量哑铃 |
| **渐进超负荷** | ✅ 容易实现（递增重量）| ❌ 很快到瓶颈 |
| **增肌效果** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐（前期有效，后期受限）|
| **动作学习** | 有镜子、有人旁指 | 只能看视频自学 |
| **月均成本** | 150-300 元/月 | 购置器材约 500-2000 元(一次性) |

> 瘦人增肌最核心的原则是「渐进超负荷」，即每周要比上周负重多一点。在家用自重训练**很快就会遇到瓶颈**（几周后俯卧撑就不再有增肌效果了）。健身房的器械可以让你**持续递增负重**，这是家庭训练难以替代的。

### 是否需要请私教？

**结论：建议买 10-15 节课，足够用。不建议长期请。**

| 阶段 | 建议 | 理由 |
|---|---|---|
| **前 1-2 个月** | ✅ **建议请私教** | 你没有任何健身经验。深蹲、硬拉、卧推等动作如果姿势错误，**非常容易受伤**（特别是腰和膝盖）。私教能帮你建立正确的动作模式。 |
| **第 3 个月起** | ❌ 不再需要 | 基础动作已掌握，后续自己按计划练即可。教练的主要价值在动作纠正和初期规划，不是长期陪练。 |

**私教选择注意事项**：
- ✅ 选有 **NSCA/ACSM/ACE** 等国际认证的教练
- ✅ 第一节课先观察：好教练会先做体态评估，而非直接让你练
- ❌ 不要买太多课（10-15 节足够学会所有基础动作）
- ❌ 警惕强制推销几十节课的销售型教练

**费用预估**：
- 私教费用：约 200-400 元/节 × 10-15 节 = **2000-6000 元（一次性投入）**
- 健身房月卡：约 150-300 元/月

## 📈 第六部分：合理预期与里程碑

> [!TIP]
> **三个月后的训练升级路径（v1→v2 完善，来源：🟡2）**：当你满足以下**全部**条件时，才考虑升级到上下肢分化（Upper/Lower Split）：
> - 深蹲重量 ≥ 自身体重 × 1.0（即 ≥ 50kg）
> - 卧推重量 ≥ 自身体重 × 0.75（即 ≥ 37.5kg）
> - 连续 8 周训练，主要动作重量每周都有进步
> - 主要动作姿势已经稳定，无需专注于"记动作"

### 新手福利期（前 3-6 个月）

初学者有一个**「新手福利期」**（Newbie Gains）— 你的肌肉从未受过力量训练刺激，会对训练产生极强的适应反应，这是增肌最快的阶段。

| 阶段 | 时间 | 预期变化 |
|---|---|---|
| **启动期** | 第 1-4 周 | 力量明显提升（主要是神经适应），体型变化不大 |
| **成长期** | 第 2-3 个月 | 开始看到肌肉轮廓，衣服开始变紧，体重增长 2-4kg |
| **爆发期** | 第 4-6 个月 | 明显的体型变化，手臂、胸肌、背部有肉，体重增长 5-8kg |
| **稳定期** | 第 7-12 个月 | 增速放缓，但体型已有质变，目标体重 58-63kg |

### 阶段目标清单（多维追踪，v1→v2 完善，来源：🟡4）

> [!NOTE]
> **追踪原则**：不要只看体重。增肌初期经常出现「体重没变但体型变了」（肌肉增加 + 体脂减少同步进行）。建议每月记录 4 个指标：**体重 + 围度（腰/胸/臂）+ 主要动作重量 + 照片对比**。

- [ ] **第 1 周**：加入健身房，预约私教体验课，完成「第 1 周操作指南」的熟悉流程
- [ ] **第 2 周**：饮食习惯调整到位（每天 6 餐，蛋白质 90g+），外食场景有方案
- [ ] **第 1 个月**：每周五做 5 分钟复盘（训练次数 / 蛋白质达标 / 体重）
- [ ] **第 1 个月末**：深蹲从空杆到 30-40kg，卧推从空杆到 25-35kg；拍围度照片对比
- [ ] **第 3 个月**：体重达到 53-55kg，深蹲 50-60kg，卧推 40-50kg；评估是否满足升级标准
- [ ] **第 6 个月**：体重达到 56-60kg，身型有明显改变；围度对比显著
- [ ] **第 12 个月**：体重达到 60-65kg，基本完成体型转变

## ⚠️ 第七部分：常见误区警告

| 误区 | 正确做法 |
|---|---|
| ❌ "先跑步减脂再增肌" | ✅ 你已经很瘦了，**直接增肌**。大量有氧会消耗你宝贵的热量 |
| ❌ "只练手臂/胸肌" | ✅ 全身均衡训练。腿部训练能刺激全身生长激素分泌，对增肌极其重要 |
| ❌ "追求大重量" | ✅ 先用轻重量把动作做标准，受伤了一切归零 |
| ❌ "练得越多越好" | ✅ 肌肉在休息时生长。过度训练反而掉肌肉 |
| ❌ "不吃脂肪" | ✅ 健康脂肪（坚果、橄榄油、鱼油）对荷尔蒙分泌至关重要 |
| ❌ "蛋白粉有害/是激素" | ✅ 乳清蛋白就是从牛奶中提取的蛋白质，完全安全 |
| ❌ "只要练就会长肌肉" | ✅ **吃不够不会长**。饮食占增肌成功的 70% |
| ❌ "体重两周没变就是没效果" | ✅ 同时追踪围度和力量，体型变化比体重更重要 |

## 🔧 第九部分：遇到问题时的诊断和调整（v1→v2 新增，来源：🔴1）

> [!IMPORTANT]
> 增肌过程中**一定会遇到障碍**，这不代表计划失败。以下是最常见问题的诊断和应对方案。

### 🍽️ 问题一：体重停滞（连续 3 周不涨）

```
体重连续 3 周未增长？
    ↓
检查：薄荷健康 App 里，过去 2 周的平均每日热量是多少？
    ↓
< 2300 大卡？ → 每天增加 200-300 大卡（加一杯牛奶 + 一个鸡蛋即可）
≥ 2300 大卡？ → 检查训练强度：每周有没有在渐进超负荷？
    ↓
训练强度没增加？ → 下次训练尝试增加 1-2.5kg 重量或多做 1 次
训练强度有增加？ → 再等 2 周观察，外胚型增肌速度本来就慢
```

### 💪 问题二：力量停滞（连续 2 周主要动作没有进步）

**按以下顺序检查：**
1. **睡眠**：平均每天有没有睡够 7 小时？睡眠不足是力量停滞最常见原因
2. **热量**：当天训练前 2-3 小时有没有吃足够的碳水？
3. **动作**：是不是重量太重导致动作变形？降 10% 重量专注动作质量
4. **恢复**：上次训练到现在有 48 小时以上吗？

### 🤕 问题三：肌肉酸痛

| 情况 | 原因 | 应对 |
|---|---|---|
| 训练后 24-48 小时酸痛 | 正常！延迟性肌肉酸痛（DOMS）| 正常训练，适当拉伸 |
| 酸痛持续 4 天以上 | 训练量过大或睡眠不足 | 延迟下次训练日 1-2 天 |
| 关节疼（尖锐刺痛）| ⚠️ 可能受伤 | 立即停止该动作，休息观察 |

### 🚨 问题四：受伤

- **局部受伤**（如肩部）：停止受伤部位的训练，其他部位正常练（如肩伤期间可以练腿）
- **严重疼痛或肿胀**：停止全部训练，就医
- **恢复期**：受伤部位完全不痛后，先用 50% 重量恢复训练，再逐步回到正常水平
- **预防优先**：任何动作感到关节不适，立即降重量或换替代动作

> ## 🔑 一句话总结
> **吃够（每天 2400 大卡 + 90g 蛋白质） + 练对（每周 3 次全身力量训练） + 睡足（每天 7-8 小时） = 增肌成功。**
> 
> 三者缺一不可，但饮食最重要。**遇到问题不要放弃，对照第九部分诊断后调整。**
