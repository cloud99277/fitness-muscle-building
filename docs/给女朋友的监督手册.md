# 增肌与身体管理监督手册 — 给女朋友版

> 这份手册是给你的。  
> 你的角色不是当严厉教练，而是帮他把计划从“想做”变成“每天真的做到一点”。  
> 最重要的监督不是骂他，而是帮他回到节奏里。

---

## 1. 你需要先了解整个计划

他的目标不是单纯练大肌肉，而是一个 **8-12 个月的身体重建计划**：

| 维度 | 当前情况 | 目标 |
|---|---|---|
| 身高体重 | 165cm / 50kg，偏瘦 | 逐步增到 60-65kg |
| 训练基础 | 零基础 | 先居家哑铃，再视情况过渡健身房 |
| 睡眠 | 日均 5-6 小时 | 先稳定到 7 小时以上 |
| 饮食 | 不规律，蛋白不足 | 先补到 75g 蛋白，再逐步到 100g+ |
| 主要问题 | 偏瘦、疲劳、睡眠不足、体能弱 | 先修复基础，再增肌 |

一句话理解：  
**先睡够、吃够、喝够，再稳定训练。**

---

## 2. 这个计划分 3 个优先级

### 第一优先级：基础修复，先坚持 14 天

这 14 天你重点盯这 3 件事：

| 项目 | 合格线 | 你怎么监督 |
|---|---|---|
| 睡眠 | 23:30 前睡，睡满 7 小时 | 晚上 22:30 后提醒他收手机、洗澡、准备睡觉 |
| 蛋白质 | 每天至少 75g | 问他今天有没有鸡蛋、牛奶、肉、蛋白粉 |
| 饮水 | 每天 2L 左右 | 问他水壶有没有喝完 |

这一阶段不要急着逼他练很猛。  
如果他睡眠还是 5-6 小时，训练越多不一定越好，先把恢复补起来。

### 第二优先级：稳定训练

基础修复能做到后，训练目标是每周 2-3 次。

当前优先执行居家哑铃训练：

| 星期 | 安排 |
|---|---|
| 周一 | 推 A：胸、肩、三头 |
| 周二 | 休息 |
| 周三 | 拉 B：背、二头 |
| 周四 | 休息 |
| 周五 | 腿 C：腿、臀、核心 |
| 周六 | 可选散步 |
| 周日 | 完全休息 |

每次训练一般 45-50 分钟。  
如果当天很累、睡眠少于 6 小时、工作或备考压力很高，不是直接摆烂，而是做 **降级训练**：

| 情况 | 应该怎么做 |
|---|---|
| 轻度疲劳 | 少做 1 个辅助动作，降低强度 |
| 很累或睡眠 <6h | 只做 2-3 个主要动作，每个 2 组，20-30 分钟结束 |
| 关节刺痛、头晕胸闷 | 停止训练，必要时就医 |

### 第三优先级：长期增肌

第 5 周以后，饮食目标会逐步提高：

| 阶段 | 热量 | 蛋白 |
|---|---|---|
| 第 1-2 周 | 2000-2200 kcal | 75g |
| 第 3-4 周 | 2200-2300 kcal | 85-90g |
| 第 5 周起 | 2300-2420 kcal | 100-110g |

所以你前期不要只用“有没有吃到 100g 蛋白”来判断他失败。  
V4.1 的核心是：**先把最低线稳定住，再逐步加量。**

## 3. 你每天需要监督什么

每天晚上花 2 分钟，看他追踪 App 或让他发一条打卡。

| 检查项 | 合格标准 | 如果没做到，你怎么说 |
|---|---|---|
| 睡眠 | 23:30 前准备睡，至少 7h | “今天先别硬撑了，早点睡比多练一组重要。” |
| 饮食 | 基础期 2000 kcal + 75g 蛋白 | “蛋白还差一点，要不要补个蛋/牛奶/蛋白粉？” |
| 饮水 | 2L 左右 | “水壶喝完了吗？差多少？” |
| 训练 | 训练日完成完整或降级训练 | “今天是完整练，还是做降级版？” |
| 久坐 | 每 45 分钟起来动一动 | “坐太久了，起来走 3 分钟。” |
| 凯格尔 | 每天 3 组 | “睡前那 3 组做了吗？” |
| 记录 | App 里有记录 | “你记录了吗？不记就不知道有没有进步。” |

最有用的问题不是“你怎么又没做到”，而是：

```text
今天卡在哪里了？
明天怎么让它更容易一点？
```

## 4. 每日打卡模板

你可以让他每天发这条：

```text
日期：
睡眠：__ 点睡，睡了 __ 小时
饮食：热量约 __ kcal，蛋白约 __ g
饮水：__ L
训练：完整训练 / 降级训练 / 休息日 / 没做
凯格尔：__ 组
今天最难的是：
明天要改的是：
```

如果他用了追踪 App，你只要看这几项：

| App 位置 | 看什么 |
|---|---|
| 仪表盘 | 平均睡眠、基础蛋白、今日饮水 |
| 训练页 | 本周训练次数，是否有“降级版”记录 |
| 更多 → 周复盘 | 本周训练、蛋白、睡眠、凯格尔达标天数 |

## 5. 饮食监督重点

他偏瘦，真正的难点通常不是练，而是 **吃不够**。

你不用帮他算得特别精细，只要盯住“蛋白质有没有落地”：

| 食物 | 大概蛋白 |
|---|---|
| 1 个鸡蛋 | 6g |
| 250ml 牛奶 | 8g |
| 1 勺蛋白粉 | 20-25g |
| 150g 鸡胸/牛肉/鱼虾 | 25-35g |
| 豆腐/豆制品一份 | 10-20g |

基础期最简单组合：

```text
2 个鸡蛋 + 2 杯牛奶 + 一份肉/鱼/虾 + 一点豆制品
```

这样通常就能接近或超过 75g。

### 你可以重点提醒

- 早餐不要空着上班。
- 午餐尽量有大块肉、蛋、豆腐。
- 外卖优先点双倍肉、少油少盐。
- 吃不下时，不要硬塞大餐，可以分多次吃。
- 蛋白粉是补充，不是代替正餐。

## 6. 睡眠监督重点

睡眠是这个计划的第一核心。

他的目标不是“每天完美”，而是先从 5-6 小时拉到 7 小时以上。

| 时间 | 你可以提醒什么 |
|---|---|
| 22:00 | 少刷短视频，准备进入睡前模式 |
| 22:30 | 洗澡、整理明天东西 |
| 22:45 | 做凯格尔 |
| 23:00 | 手机远离床，阅读/放松 |
| 23:30 | 关灯睡觉 |

如果他晚上还在刷手机，可以直接提醒：

```text
你现在不是缺意志力，是缺睡眠。先睡觉，明天再变强。
```

## 7. 训练监督重点

你不需要记住每个动作怎么做，只要知道训练日和原则。

| 训练日 | 主要练什么 | 你问什么 |
|---|---|---|
| A 推 | 胸、肩、三头 | “今天推日做了吗？” |
| B 拉 | 背、二头 | “背部有没有发力感？” |
| C 腿 | 腿、臀、核心 | “腿练完有没有太痛？” |

训练最重要的不是拼命，而是：

```text
动作标准 + 每周一点点进步 + 不受伤
```

### 什么情况不要逼他完整训练

- 睡眠少于 6 小时。
- 明显头晕、胸闷、恶心。
- 关节尖锐疼痛。
- 工作/备考当天消耗很大。
- 情绪明显崩掉。

这种时候请让他做降级版，或者散步 20 分钟。

## 8. 关于私密健康和性功能部分

计划里有一部分是性功能重建，包括凯格尔、睡眠、减压、减少高频刺激、行为训练等。

你需要知道的是：

- 这不是“他不行”，也不是道德问题。
- 它更像一个神经和身体状态的重建过程。
- 睡眠不足、压力大、频繁刷刺激内容，都会影响恢复。
- 你的支持重点是减少焦虑，不是制造表现压力。

你可以监督的部分：

| 内容 | 怎么监督 |
|---|---|
| 凯格尔 | 每天问一句“3 组做了吗？” |
| 睡眠 | 帮他早点下线、早点睡 |
| 数字戒断 | 晚上少刷短视频/色情刺激/无意义信息流 |
| 焦虑 | 不用羞辱、比较、催促表现 |

### 需要就医的情况

如果长期困扰，或者伴随勃起困难、疼痛、尿路症状、明显焦虑，或者自我训练 8-12 周没有改善，建议去男科或泌尿外科评估。

这不是丢脸，是对身体负责。

## 9. 每周你们可以一起复盘一次

每周五或周日，花 5 分钟问这 6 个问题：

| 问题 | 合格参考 |
|---|---|
| 本周训练几次？ | 2-3 次即可 |
| 蛋白达标几天？ | 基础期至少 5 天 |
| 睡眠达标几天？ | 至少 5 天 |
| 饮水达标几天？ | 至少 5 天 |
| 凯格尔做了几天？ | 至少 5 天 |
| 体重/精神状态怎样？ | 先看趋势，不看一天波动 |

复盘时不要只问“为什么没做到”。更好的问法是：

```text
这周哪个环节最难？
下周我们只改一件事，改哪件？
我怎么提醒你最有效？
```

## 10. 你最需要注意的风险

### 训练风险

| 信号 | 处理 |
|---|---|
| 肌肉酸痛 24-48 小时 | 正常 |
| 酸痛超过 4 天 | 训练量太大，降级 |
| 关节刺痛 | 停练 |
| 头晕、胸闷、异常心慌 | 停止训练，必要时就医 |

### 饮食风险

| 信号 | 处理 |
|---|---|
| 腹胀、腹泻 | 降低蛋白粉单次用量，分多次吃 |
| 反复腹泻或体重下降 | 考虑乳糖不耐受或就医 |
| 连续 3 周体重不涨 | 先查热量和睡眠，再加餐 |

### 心理风险

| 信号 | 处理 |
|---|---|
| 因为没完成而自责 | 帮他回到明天的最小行动 |
| 因为训练焦虑 | 允许降级训练 |
| 因为私密问题焦虑 | 降低表现压力，必要时专业评估 |

## 11. 监督方式：该做和不该做

### 你可以做

- 每天晚上提醒他记录。
- 训练日问他做完整训练还是降级训练。
- 睡前帮他下线，不让手机拖走睡眠。
- 他没做到时，问原因和明天怎么调整。
- 他做到时，明确夸具体行为，比如“你今天睡眠和蛋白都达标了”。

### 尽量不要做

- 不要用羞辱、比较、威胁的方式监督。
- 不要把一次没做到上升成“你就是不自律”。
- 不要在他明显疲劳时强迫完整训练。
- 不要把体重一天的波动看得太重。
- 不要把私密健康问题变成压力源。

---

## 12. 最简监督口诀

```text
先睡够，再吃够。
能完整练就完整练，累了就降级练。
每天记录，周末复盘。
不羞辱，不摆烂，只回到下一步。
```

你真正要帮他的，不是每一天都完美。  
是让他在没做到的时候，也能重新回到计划里。
