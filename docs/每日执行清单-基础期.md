## 每日时间轴（训练日版）

```
07:30  起床
       └─ 早餐：2个鸡蛋 + 牛奶/豆浆 + 碳水（面包/粥/红薯）

08:00  上班
       └─ 工位放好 1.5L 水壶

10:00  加餐：坚果一把 + 水果
       └─ 检查水壶进度

12:00  午餐（食堂防御策略）
       └─ 选清蒸/白灼/大块肉 + 粗粮替代一半白米饭
       └─ 重油菜过水涮

15:30  加餐：香蕉 + 鸡蛋

17:00  下班

17:30  训练前加餐：面包/香蕉（碳水）

18:00  🏋️ 开始训练（45-50 分钟）
       └─ 看下面的「训练执行卡」

19:00  训练后：1勺蛋白粉 + 高蛋白晚餐

20:00  个人时间

22:30  睡前流程开始
       ├─ 洗澡
       ├─ 凯格尔 3 组（收缩5秒→放松5秒×10次）
       ├─ 准备明天衣物
       └─ 手机远离床

23:30  关灯睡觉
```

## 每日时间轴（休息日版）

```
同上，跳过 18:00 训练，改为散步 20 分钟
每 45 分钟起身走动 3-5 分钟
```

## 今日必做清单（每天打勾）

```
□ 早餐吃了（不空腹上班）
□ 蛋白质达标（基础期 ≥75g）
□ 饮水 ≥2L（水壶喝完）
□ 训练日：完成训练 / 休息日：散步 20 分钟
□ 凯格尔 3 组
□ 23:30 前上床
□ App 记录（体重/训练/睡眠/饮水至少记一项）
```

## 基础期营养速查

| 食物 | 蛋白 | 怎么吃 |
|------|------|--------|
| 1 个鸡蛋 | 6g | 早餐 2 个 = 12g |
| 250ml 牛奶 | 8g | 早晚各 1 杯 = 16g |
| 1 勺蛋白粉 | 20-25g | 训练后 1 勺 |
| 150g 鸡胸/牛肉 | 25-35g | 午餐或晚餐 |
| 豆腐一份 | 10-20g | 随餐 |

**基础期最低组合**（确保达标）：
```
2蛋(12g) + 2奶(16g) + 1肉(30g) + 蛋白粉(22g) = 80g ✅
```

## 降级训练规则

| 状态 | 怎么练 |
|------|--------|
| 正常 | 完整训练 45-50 分钟 |
| 轻度累 | 少做 1 个辅助动作 |
| 很累/睡<6h | 只做 2-3 个主动作，每组 2 组，20-30 分钟 |
| 关节痛/头晕 | 停练，散步即可 |

---

*基于 V4.1 基础修复期标准：2000kcal + 75g蛋白 + 7h睡眠*
*第 15 天起切换到正常期清单（2300kcal + 100g蛋白）*
