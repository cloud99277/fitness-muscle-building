## 一、定位与愿景

### 一句话定义

为一名 165cm / 50kg / 28 岁 / 零基础 / 早8晚5双休男性，构建一套**全栈身体管理系统**，覆盖增肌训练、内分泌修复、性功能重建、作息重塑四大维度，配套科学量化追踪应用。

### 项目目标

| 维度 | 目标 |
|---|---|
| **体重目标** | 60-65kg（增重 10-15kg，净增肌约 5-8kg） |
| **时间线** | 8-12 个月完成体型转变 |
| **执行标准** | 有具体的每日饮食方案、每周训练计划、月度里程碑 |
| **科学性标准** | 每个核心建议有明确的科学依据（Mifflin-St Jeor / 渐进超负荷 / 蛋白质合成窗口等） |

### 与通用增肌方案的差异

| 维度 | 通用方案 | 本项目 |
|---|---|---|
| 体质 | 普通体型 | 外胚型（新陈代谢快，增肌难度高） |
| 经验 | 假设有基础 | 零基础（需学习基础动作 + 私教辅助期） |
| 时间约束 | 无特殊 | 早8晚5双休，晚上健身房 17:30-18:45 |
| 预算约束 | 无特殊 | 月均 580-1050 元预算 |
| 目标 | 增肌塑形 | 增肌 + 建立健康习惯（可持续性优先） |

### 独特优势与代价（诚实分析）

**优势：**
- 新手福利期效果显著，前 3-6 个月增长最快
- 从零开始，无需纠正错误动作习惯
- 目标体重合理可达（10-15kg 增重）

**代价：**
- 外胚型体质需要更高热量盈余（比普通初学者难吃够）
- 初学者动作错误风险高 → 需私教辅助期（一次性 2000-6000 元）
- 饮食习惯改变难度大（从随意饮食到每日 6 餐计划饮食）

## 三、外部洞察

### 增肌科学核心依据

| 原则 | 科学基础 | V1 是否覆盖 |
|---|---|---|
| 热量盈余 | 能量守恒 + 肌肉合成需要原料 | ✅ TDEE + 380-500 大卡 |
| 蛋白质摄入 | Meta 分析：≥1.6g/kg 体重/天（Morton 2018） | ✅ 100-110g/天 |
| 渐进超负荷 | 肌肉适应原理（每周微增负荷） | ✅ 作为核心法则 |
| 复合动作优先 | 多关节动作触发更多肌群 + 更大自然激素响应 | ✅ 深蹲/卧推/硬拉为核心 |
| 睡眠质量 | 生长激素在深度睡眠中分泌高峰（睡前 1h 内） | ✅ 7-8 小时 |
| 训练量适配 | 初学者对高训练量适应差，全身训练 3 次/周最优 | ✅ 3 天 A/B/C |

### 引力陷阱过滤（被砍掉的建议）

> **引力陷阱**：看起来合理但实际上不适合当前约束条件的建议

| 被砍建议 | 看起来合理 | 为什么砍 |
|---|---|---|
| 推拉腿 PPL（6天/周）| 训练量大、效果好 | 零基础者恢复能力和技术储备不足；早8晚5留给训练的时间有限 |
| 大量有氧（跑步减脂）| 看起来塑形效果好 | 外胚型不需要减脂，大量有氧消耗宝贵热量，抑制增肌 |
| 长期请私教 | 有指导效果更好 | 性价比低，基础动作 10-15 节课可以学完，后续无需持续花费 |
| 高级补剂（增肌粉/氨基酸/支链氨基酸等） | 补剂越多效果越好 | 初学者只需蛋白粉+肌酸，其他补剂对初学者效果边际递减 |

## 五、设计约束

这些约束条件是不可协商的，任何方案修改都不能违背：

| 约束 | 具体值 | 影响 |
|---|---|---|
| **时间** | 早8晚5，17:30-18:45 训练窗口 | 训练时间固定 ~75 分钟 |
| **频率上限** | 每周最多 3-4 次训练 | 不适合 PPL 等高频方案 |
| **体质** | 外胚型，BMI 18.4 | 热量必须盈余，禁止有氧为主  |
| **经验** | 零基础 | 前 1-2 个月必须有私教辅助 |
| **预算** | ~580-1050 元/月（含健身房+饮食+补剂） | 不适合高端补剂或大量私教课 |
| **饮食习惯** | 上班族，午饭在外/食堂 | 午餐需要可外食的高蛋白选项 |

## 六、不做什么（Anti-Patterns）

> 这些是从科学原则和个人约束中提炼的明确禁区：

| ❌ 不做 | ✅ 对应的正确做法 | 原因 |
|---|---|---|
| ❌ 大量有氧（跑步/HIIT） | ✅ 每周最多 1-2 次，20 分钟轻度 | 外胚型不需要减脂，有氧消耗热量 |
| ❌ 只练胸肌手臂 | ✅ 全身均衡 + 腿部训练 | 腿训促进全身睾酮/生长激素分泌 |
| ❌ 追求大重量 | ✅ 8-12 次 / 组，动作先于重量 | 零基础受伤风险高 |
| ❌ 天天练 | ✅ 隔天练，休息时肌肉才生长 | 过度训练反而掉肌肉 |
| ❌ 不吃脂肪 | ✅ 70-80g/天健康脂肪 | 脂肪是睾酮合成的原料 |
| ❌ 先减脂再增肌 | ✅ 直接进入增肌阶段 | BMI 18.4，无需减脂 |
| ❌ 长期靠补剂代替真实食物 | ✅ 蛋白粉 + 肌酸作为补充，食物为主 | 整体饮食质量更重要 |
